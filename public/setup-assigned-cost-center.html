🚀 BonBox Setup

Kostenträger-Spalte erstellen

⚠️ WICHTIG! Die Kostenträger-Spalte existiert NOCH NICHT in der Datenbank. Deswegen werden Ihre Kostenträger-Auswahlen nicht gespeichert. Bitte führen Sie diese SQL-Befehle SOFORT aus:
1

SQL-Editor öffnen

Klicken Sie auf den Button, um den Supabase SQL-Editor zu öffnen:

2

SQL kopieren und einfügen

Kopieren Sie diese SQL (klicken Sie zum Kopieren):

ALTER TABLE receipt_items ADD COLUMN IF NOT EXISTS assigned_cost_center_id uuid REFERENCES cost_centers(id) ON DELETE SET NULL;

Klick zum Kopieren ↑

3

In den Editor einfügen und ausführen

1. Fügen Sie die SQL in den SQL-Editor ein (großes Textfeld)
2. Drücken Sie Ctrl+Enter (Windows) oder Cmd+Enter (Mac)
3. Sie sollten eine Erfolgsmeldung sehen

4

Zurück zu BonBox

Laden Sie die BonBox-App neu und probieren Sie die Kostenträger-Auswahl:

✓ Erfolgreich! Die Spalte wurde erstellt. Sie können jetzt Kostenträger auswählen!
